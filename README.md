# Notes Management Application

A full-stack Notes Management Application developed using the MERN Stack.

## Features

* Add Notes
* View Notes
* Delete Notes
* REST API Integration
* MongoDB Database Storage
* Responsive UI

## Tech Stack

### Frontend

* React.js
* HTML5
* CSS3
* JavaScript
* Axios

### Backend

* Node.js
* Express.js

### Database

* MongoDB
* Mongoose

## Installation

### Backend

```bash
cd backend
npm install
npx nodemon server.js
```

### Frontend

```bash
cd frontend
npm install
npm start
```

## Project Description

This project demonstrates CRUD operations using React.js, Node.js, Express.js, and MongoDB with REST API integration.
